본 문서는 설정 파일 예시 문서입니다.

### 경로

- server/src/config

### 파일

1. config.json

```js
{
  "host": "데이터베이스 호스트",
  "user": "데이터베이스 유저",
  "password": "데이터베이스 유저에 해당하는 패스워드",
  "database": "데이터베이스명"
}
```

2. awsconfig.json

```js
{
  "accessKeyId": "aws 액세스 키 id",
  "secretAccessKey": "aws 시크릿 액세스 키 값",
  "region": "리전"
}
```
